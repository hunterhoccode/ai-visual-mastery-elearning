## 1. UI Prompts

**General Prompt**

Design a compact, modern UI card inside a 1x1 frame. Choose a specific feature or function (e.g. file sharing, transaction list, dashboard). Describe the layout clearly (e.g. vertical card, hover states, shadows, icons), use distinct color palettes (grays, dark themes, neon accents), and define the text structure (title, subtitle, tooltips). Include interaction logic like dropdowns, profile images, metrics, or notifications. Subtle design details like gradients, radial noise, glassmorphism, or soft elevation can enhance realism. End with layout and purpose hints.

**Suggested Keywords:**

glassmorphic, neon accents, vertical card, crypto dashboard, hover tooltip, radial noise, transaction feed, compact UI, minimal layout, mobile-friendly

One compartment contains a glowing waveform visual in rainbow gradients (magenta, yellow, cyan), while another features a hover-highlighted share icon with a multicolor gradient stroke. ## 2. Micrographic Prompts

**General Prompt**

Design a 1x1 micrographic badge or diagnostic token layout. Stick to black on off-white or grayscale color schemes. Use structured, grid-based design and dense informational blocks: terminal logs, codes, tags, warnings. Layout should include monospaced or uppercase type, layered sections, framed scroll areas, and strict hierarchy. Imperfections are essential: barcode misprints, toner dust, paper folds, dot-matrix fading, or offset overlays. These prompts simulate high-density archival visuals with a fictional/technical aesthetic.

**Suggested Keywords:**

token badge, transmission log, security label, offset print, barcode glitch, terminal screen, dot-matrix, heuristic archive, overlay label, fold shadow

### Example Prompts

**Prompt 1:**

Design a densely packed 1x1 diagnostic token badge. Use grid-based layout, black text on off-white. Top: "STATIC RETURN / NODE 4H" in uppercase extended font. Add file-type string:

"TYPE: HEURISTIC.ARC"

"NODE AUTH: ΔX-04-98"

Mid-section block: frame with tiny scroll text:

"TRANSMISSION LOG: → 08.09.25 22:42 — INIT // → 08.09.25 22:44 — FREQ MATCHED → 08.09.25 22:45 — CORE RETURNED (ERROR 409) → 08.09.25 22:47 — SYNC FAILED / END"

Bottom features security tags:

"PROPERTY OF / ARCHIVUM 9" "DO NOT RESCUE" "NOTHING IS LOST IF NOTHING RETURNS"

Imperfections: barcode with overprinted text, misaligned framing box, torn scan edge along lower left, dot-matrix fade, light vertical stripe haze through top half.

**Prompt 2:**

Design a 1x1 overlay mimicking a redirection interface label. Core text: "REDIRECT NODE_08", inside large outlined triangle pointing right. Add:

"PASS KEY: 44–213" "JUMP SET FOR 03.21.24" "STATUS: PRE-LOCK"

Use faded red for triangle, black for text. Include tiny arrows and dotted motion path.

Imperfections: triangle outline double-printed with offset, paper looks buckled, toner powder residue along path line, faint "shadow print" of triangle repeating behind the first.

**Prompt 3:**

Design a gravity regulation seal in halftone print style. Title: "GRAV UNIT 93" in compressed grotesk, placed inside an oval with dot-fade texture. Background: raw white paper with diagonal measurement ticks. Include phrases like:

"CERTIFIED FOR ZONE 3-B" "MAX LOAD: 980 N" "REF MODULE: GNX.440A"

Imperfections: halftone circles offset from text, warping ink ripple on main logo, faint fold shadow in lower-left, toner spill that bleeds outward slightly, two stray dots where no marks should be.

## 3. Retro Logo Prompts

**General Prompt**

Create a retro-style vector logo reinterpretation of an existing or fictional brand. Focus on thick typography (bubble letters, geometric sans-serif), flat fills, bold outlines, and heavy drop shadows. Add sparkles, retro ornaments, or motion lines for personality. The symbol (if any) should stay recognizable but repositioned creatively. Background should always be flat beige or transparent – no gradients or 3D effects. Keep it playful, bold, and clean.

**Suggested Keywords:**

retro vector, drop shadow, bubble type, badge layout, sticker logo, sparkles, 80s font, comic accents, geometric sans, flat design

### Example Prompts

**Prompt 1:**

A retro-styled vector logo for the brand "ohneis," The word "ohneis" is the core visual focus, redrawn in ultra-bold lowercase geometric lettering, slightly softened and exaggerated, reminiscent of 90s sticker branding and 70s type-driven logos. Use solid black fills with cream outlines and a thick black drop shadow for punchy, sticker-like depth.

Scattered around the wordmark, add small retro starbursts and sparkle stars — some sharp, some round — in matching monochrome or a pop accent like mint green or faded yellow. The composition can be loosely badge-style or horizontally aligned, but should feel vibrant, fun, and high-contrast.

Everything must remain flat, vector-based, and on a transparent background.

**Prompt 2:**

A retro-style vector reinterpretation of the ChatGPT logo, preserving the green knot symbol exactly as-is. The symbol is reduced and moved to the side or behind the text. The word "ChatGPT" is scaled up and redrawn in a bold, all-caps geometric sans-serif, reminiscent of 80s computing and sci-fi titles. Use a deep green for the fill and add black outlines plus a strong drop shadow for dimension. Optional accent: a bright green spark or radial burst behind the type. Layout is badge-style or staggered horizontal, always flat and clean. Background is light beige or transparent.

**Prompt 3:**

A retro vector reinterpretation of the Gatorade logo, keeping the original lightning bolt and bold "G" icon completely intact. Move the lightning emblem smaller and to the top or side, and make the word "GATORADE" the center of attention. Use explosive chunky bubble letters, with a mix of orange and yellow fills, outlined thick in black and layered with a heavy drop shadow. Add comic-style sparks or minimal motion lines around the text for extra energy. Layout is horizontal or circular badge. Background is light beige or transparent. Flat and clean — no gradients or textures.
